Privacy Policy for Java OOP Bangla App

Last updated: April 17, 2025

📱 Overview

The Java OOP Bangla app is developed to help users learn Object-Oriented Programming in Java in the Bengali language. Your privacy is important to us. This policy explains how we handle your data.

🔐 Data Collection

We do not collect any personal data from users. The app functions fully offline and does not require any login, signup, or internet access for learning purposes.

📦 Third-Party Services

This app does not use any third-party services (such as Google Firebase, Google Analytics, Facebook SDK, etc.) that collect user information.

🛠️ Open-Source Libraries

To build this app, we use several open-source libraries such as:

  • React Native
  • React Navigation
  • Other utility and UI-focused libraries

These libraries are only used to support app functionality and do not collect or transmit personal data.

🔒 Security

Since we do not collect or store any user data, there is no risk of your data being shared or misused. The app does not access sensitive device information.

👶 Children's Privacy

This app does not target children under the age of 13, and we do not knowingly collect any data from children.
